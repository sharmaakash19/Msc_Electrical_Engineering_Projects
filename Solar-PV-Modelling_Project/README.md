# Solar PV Modelling 

This project was part of the Masters:

It includes:

1. Single-diode photovoltaic modelling in MATLAB/Simulink
2. I–V and P–V curve extraction and comparison with experimental data
3. Nominal Operating Cell Temperature (NOCT) performance analysis


##  Solar PV Single Diode Modelling

A complete single-diode model of the SW85 PV module was developed in MATLAB/Simulink.

Key features:
- Temperature-dependent photocurrent modelling
- Reverse saturation current calculation
- Thermal voltage computation
- I–V and P–V curve generation
- Experimental comparison validation

---

## PNOCT Performance Evaluation

Simulation was performed under:
- 800 W/m² irradiance
- 46°C temperature (NOCT)
- Air mass 1.5

Results were compared with manufacturer data.
